Terminal - Projeto 1: Jogo Joquempô

Projeto: Raciocínio Algorítmico - Jogo Joquempô

Descrição:

Este projeto é um jogo Joquempô desenvolvido em Python como parte da disciplina Raciocínio Algorítmico. O jogo possui três modos de jogo: jogador vs jogador, jogador vs computador, e computador vs computador. Foram utilizadas funções, estruturas de desvio, dicionário e a biblioteca de randomização.

Características principais:

  • Modo de jogo: Jogador vs Jogador
  • Modo de jogo: Jogador vs Computador
  • Modo de jogo: Computador vs Computador
  • Implementação de lógica com funções e estruturas de desvio
  • Utilização de dicionário para armazenar os resultados
  • Randomização para jogadas do computador

Desafios enfrentados:

  • Implementação de diferentes modos de jogo
  • Garantir a aleatoriedade das jogadas do computador

Ferramentas utilizadas:

  • Python
  • Biblioteca random
