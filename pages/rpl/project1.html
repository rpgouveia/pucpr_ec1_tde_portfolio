Projeto: Resolução de Problemas com Lógica Matemática - Jogo Delta-9X

Descrição:

Este projeto é um jogo de investigação desenvolvido em Python como parte da disciplina Resolução de Problemas com Lógica Matemática. No jogo, um agente espacial investiga o desaparecimento de sua irmã numa colônia espacial chamada Delta-9X. Cada fase do jogo apresenta uma proposição lógica, e as dicas são hipóteses que ajudam o jogador a formular uma dedução de Verdadeiro ou Falso. Na fase final, o jogador usa os 6 valores lógicos deduzidos para resolver uma expressão lógica e determinar se o suspeito é culpado ou inocente.

Características principais:

  • Jogo baseado em proposições lógicas e deduções
  • Dicas formuladas como hipóteses lógicas
  • Fase final com resolução de expressão lógica validada por tabela verdade
  • Enredo envolvente com contexto espacial

Desafios enfrentados:

  • Desenvolvimento de proposições lógicas desafiadoras
  • Implementação da fase final com expressão lógica complexa

Ferramentas utilizadas:

  • Python
