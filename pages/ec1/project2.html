Projeto 2: Aplicativo Multimídia - Beatmaker

Descrição:

O "Beatmaker" é um aplicativo multimídia desenvolvido como parte da disciplina Experiência Criativa. Este projeto permite aos usuários criar e mixar suas próprias batidas musicais de maneira intuitiva e interativa.

Características principais:

  • Interface amigável e fácil de usar
  • Variedade de instrumentos e efeitos sonoros

Desafios enfrentados:

  • Aprendizagem da linguagem programação Java
  • Integração de áudio em tempo real
  • Otimização da performance para diferentes dispositivos

Ferramentas utilizadas:

  • Processing IDE
