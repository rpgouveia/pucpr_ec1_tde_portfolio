Projeto 3: Sistema Inteligente - Cat Care

Descrição:

O "Cat Care" é um protótipo de sistema inteligente de monitoramento desenvolvido como parte da disciplina Experiência Criativa. Este projeto visa monitorar gatinhos resgatados e alocados em clínicas parceiras da ORG Crazy Cat Gang, garantindo seu bem-estar e facilitando o acompanhamento de sua saúde.

Características principais:

  • Cadastro de usuários, gatinhos e clínicas parceiras
  • Monitoramento dos gatinhos alocados
  • Relatórios de saúde, agendamento e notificações
  • Integração com clínicas parceiras

Desafios enfrentados:

  • Aprendizagem de UI/UX
  • Aprendizagem de linguagens HTML e CSS

Ferramentas utilizadas:

  • Visual Studio Code
  • Excalidraw
  • Figma
